## 📌 Project Details

This project presents a "data-driven analysis of the PM-KISAN Scheme" to identify district-level enrollment gaps and the socio-economic factors influencing beneficiary exclusion.
Using data from the "PM-KISAN Dashboard and Census 2011", the study quantifies the uptake gap and highlights high-priority intervention areas across India.

The analysis employs "Exploratory Data Analysis (EDA), correlation analysis, geospatial visualization, and district segmentation techniques" to understand how 
literacy, banking access, and digital connectivity impact scheme enrollment. Based on these insights, districts are classified into four actionable categories—
Banking Desert, Awareness Gap, Digital Divide, and Last Mile**—to support targeted policy interventions.

The project integrates "Python, Pandas, Matplotlib, and Folium" to develop interactive visualizations and an analytical dashboard, demonstrating how data science can be leveraged for 
"evidence-based public policy and welfare governance".
